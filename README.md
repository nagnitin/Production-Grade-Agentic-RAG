# Agentic RAG Platform

Production-grade, enterprise-scale, cloud-native Agentic RAG platform.

## Features
- LangGraph Orchestration
- FastAPI Backend API
- Streamlit User Interfaces
- Qdrant Vector Store
- FlashRank Reranker
- Postgres Memory
- NeMo Guardrails
- Prometheus & LangSmith Observability

## Quickstart
To run the app locally, run:
```bash
docker-compose up --build
```
